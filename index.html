Convert String/Text into a sequence of numerical characters

use this tool to encrypt special or sensitive information in plain sight.

Let's Get strted

  1. import numero var numero = require('./numero');

  2. Initialise numero object var sequencer = new numero();

  3. Convert your string

    var mypassword = 'saf3key@word!';

    var converted = sequencer.convert(mypassword);

    The result will be like this: 70102580+++041023090@08105530697020! but will be unique for each time you convert

  4. You can change this by using the revert method var reverted = sequencer.revert("70102580+++041023090@08105530697020!")

This module can come in handly as an extra security measure for sensitive information
